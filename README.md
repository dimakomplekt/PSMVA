# **Приложение для анализа видео процесса плазменного напыления**

Приложение на базе CPP, OpenCV2 и SDL3 для анализа высокоскоростной видеосъемки плазменного напыления (расчёт характеристик различных составляющих потока).

___

### Позволяет настроить маски OpenCV, обрабатывает и формирует отчёт по форме:

| Параметр                                      | Режим 1 | Режим 2 |   ...   | Режим N |
|-----------------------------------------------|---------|---------|---------|---------|
| Средняя интенсивность излучения               |         |         |         |         |
| Средняя амплитуда струи                       |         |         |         |         |
| Средний угол направления частиц               |         |         |         |         |
| Средний угол отклонения частиц                |         |         |         |         |
| Скорость частиц в основном направлении        |         |         |         |         |
| Скорость отклоненных частиц                   |         |         |         |         |
| Процент частиц с выраженным отклонением       |         |         |         |         |
| Число Рейнольдса                              |         |         |         |         |
